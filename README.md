 Application : 7tv.de (Web App)
 Automation Tool : Selenium
 Test Framework : TestNG
 Scripting language : Java
 Project : Maven

 Selenium - For web app automation
 TestNG - For Testing framework (TDD Framework)

 Execution Engine : TestNG
